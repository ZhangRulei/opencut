# 虾剪 AI混剪平台 — UI 设计规范

> 版本：v1.1　更新日期：2026-04-09

---

## 一、设计原则

轻量、克制的现代 SaaS 风格。浅灰背景为底，白色/半透明卡片浮于其上，通过阴影和边框营造层次感，而非依赖强烈色彩对比。唯一的品牌强调色为虾红色（`#ef5d4e`），仅用于关键交互和品牌标识，其余选中/激活状态统一使用灰色系，保持视觉克制。

---

## 三、字体系统

- **字体栈**：`Inter` → `-apple-system` → `PingFang SC` → `Microsoft YaHei`
- **基础字号**：`13px`，行高 `1.5`
- **字号规格**：`10px / 11px / 12px / 13px / 14px / 15px`
- **字重**：`400` 正文 / `500` 强调 / `600` 标题
- **字间距**：`-0.01em` 全局微收紧

---

| 场景 | 圆角 |

## 五、组件规范

### 输入框

- 背景：`--bg-elevated`，`backdrop-filter: blur(16px)`
- 边框：`1px solid --border-color`，focus 时描边 `rgba(239,93,78,0.25)` + `box-shadow: 0 0 0 3px rgba(239,93,78,0.1)`

### 按钮

- **主操作按钮**：高度 `34px`，圆角 `8px`，品牌色背景，白色文字，`box-shadow: 0 4px 12px --accent-glow`
- **次要按钮**：透明背景，`1px solid --border-color`，hover 时背景 `--bg-hover`
- **禁用态**：`opacity: 0.38`，`cursor: not-allowed`

### 弹窗

- 遮罩：`rgba(0,0,0,0.36)`，`backdrop-filter: blur(2px)`
- 主体：`--bg-card`，圆角 `14px`，`--shadow-lg`，`1px solid --border-color`
- 结构：Header / Body / Footer（取消 + 确认）

### 选中状态（非品牌色场景）

功能性选中（比例、分辨率、音色等）统一灰色系：背景 `--bg-active`，边框 `--border-strong`，文字 `--text-primary`。

### 表格

- 表头：`--bg-hover` 背景，`--text-muted` 文字，`11px`，全大写，`letter-spacing: 0.04em`
- 行 hover：`--bg-hover`
- 背景：`--bg-elevated` + `backdrop-filter: blur(12px)`，圆角 `10px`

### 标签/胶囊

- 状态标签：绿色（成功）/ 红色（失败/禁用），半透明背景 + 同色边框
- 功能标签：灰色系，`--bg-active` 背景，`--text-secondary`

## 六、背景处理

页面底色使用径向渐变叠加，营造轻微空间感：

```css
background:
  radial-gradient(1000px 460px at 78% 110%, rgba(239,93,78,0.08), transparent 62%),
  radial-gradient(840px 320px at 4% -10%, rgba(17,20,28,0.04), transparent 58%),
  var(--bg-primary);
```

右下角品牌色极淡晕染，左上角极淡压暗，整体不抢眼。

---

## 七、动效规范

- **过渡时长**：`0.15s`（大多数交互）、`0.2s`（较大元素）
- **缓动**：`ease`
- **原则**：动效服务于信息传达，不做装饰性复杂动画；克制优于丰富
