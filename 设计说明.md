# UI 设计说明

## 一、设计风格

轻量、克制的现代 SaaS 风格。整体以浅灰背景为底，白色/半透明卡片浮于其上，通过阴影和边框营造层次感，而非依赖强烈色彩对比。唯一的品牌强调色为虾红色（`#ef5d4e`），仅用于关键交互和品牌标识，其余选中/激活状态统一使用灰色系，保持视觉克制。

---

## 三、字体系统

- 字体栈：`Inter` → `-apple-system` → `PingFang SC` → `Microsoft YaHei`
- 基础字号：`13px`，行高 `1.5`
- 字号规格：`10px / 11px / 12px / 13px / 14px / 15px`
- 字重：`400`（正文）、`500`（强调）、`600`（标题）
- 字间距：`-0.01em`（全局微收紧）

---

## 四、圆角规范

| 场景 | 圆角 |
|---|---|
| 页面级卡片、弹窗 | `12px ~ 14px` |
| 按钮、输入框、标签 | `8px ~ 10px` |
| 小标签、角标 | `4px ~ 6px` |
| 头像、圆形按钮 | `50%` |
| Sidebar 导航项 | `12px` |

---

## 五、组件规范

### 输入框（InputBar）
- 背景：`--bg-elevated`，`backdrop-filter: blur(16px)`
- 边框：`1px solid --border-color`，focus 时描边 `rgba(239,93,78,0.25)`
- 输入区默认高度：`56px`，随内容自动向上扩展，最大 `320px`
- 工具栏按钮高度：`34px`
- 发送按钮：`40×40px` 圆形，激活时填充品牌色并带发光阴影

### 按钮（flat-btn）
- 高度 `34px`，圆角 `8px`，边框 `1px solid --border-color`
- hover：背景 `--bg-hover`，边框加深
- 生成类型按钮（accent）：文字用品牌色，无背景填充

### 弹窗（Popover）
- `position: fixed`，跟随触发元素定位
- 背景：`--bg-elevated`（不透明，无 `backdrop-filter`）
- 阴影：`0 8px 32px rgba(0,0,0,0.12), 0 2px 8px rgba(0,0,0,0.08)`
- 圆角：`12px`

### 选中状态（非品牌色场景）
比例选择、分辨率选择、音色情感选择等功能性选中状态统一使用灰色系：
- 背景：`--bg-active`
- 边框：`--border-strong`
- 文字：`--text-primary`

### 消息头部（message-meta）
- avatar（24×24）+ prompt 文字 + tags 内联排列
- prompt 换行显示，tags 紧跟文字末尾（`inline-flex`）
- tags 字号 `11px`，灰色胶囊样式

### Sidebar
- 宽度：`76px`，固定左侧
- 导航项：`54×54px`，圆角 `12px`
- 激活态：仅文字和图标变为品牌色，无背景填充
- 底部：暗色切换按钮 + 用户头像

---

## 六、暗色模式

通过 `body.dark` 切换，所有 token 重新映射：
- 背景变为深色（`#090a0d` 系列）
- 边框改为白色低透明度
- 文字反转为浅色
- 品牌色保持不变，`accent-soft` 和 `accent-glow` 透明度略微提升
- 暗色切换按钮变为金黄色（`#f6c90e`）

---

## 七、动效规范

- 过渡时长：`0.15s`（大多数交互）、`0.2s`（较大元素）
- 缓动：`ease`
- 禁止使用 `transform: scale` 以外的位移动画用于功能性交互
- 品牌 orb 动画：`orbPulse 3s ease-in-out infinite` + `arcSpin 2.4s linear infinite`
- 流式光标：`cursorBlink 0.8s ease-in-out infinite`
- 图片生成骨架：`shimmer 1.6s linear infinite`

---

## 八、背景处理

页面底色使用径向渐变叠加，营造轻微的空间感：
```css
background:
  radial-gradient(1000px 460px at 78% 110%, rgba(239,93,78,0.08), transparent 62%),
  radial-gradient(840px 320px at 4% -10%, rgba(17,20,28,0.04), transparent 58%),
  var(--bg-primary);
```
右下角有品牌色的极淡晕染，左上角有深色的极淡压暗，整体背景不抢眼。
